# Plane wars (飞机大战)

## Built With
- Python==3.6
- pygame==1.9.4


## Operations
- :point_up: KEY_UP
- :point_down: KEY_DOWN
- :point_left: KEY_LEFT
- :point_right: KEY_RIGHT
